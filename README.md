# FUJI ROCK FESTIVAL '26 10分刻みタイムテーブル

A4横1枚で見られるように整理した、FUJI ROCK FESTIVAL '26 の日別タイムテーブルPDFを掲載するためのGitHub Pages用ファイル一式です。

## 公開方法

1. GitHubで新規リポジトリを作成します。
2. このフォルダ内の `index.html` と `assets` フォルダをアップロードします。
3. リポジトリの Settings → Pages を開きます。
4. Branch を `main`、フォルダを `/root` にして Save します。
5. 数分後に表示されるURLへアクセスします。

## 注意

これは個人用に整理した非公式タイムテーブルです。出演時間などは必ず公式発表も確認してください。
